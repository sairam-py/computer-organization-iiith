Cache Organizations
Introduction

A cache is a high speed memory that sits transparently between the processor and main memory buffering instructions or data that the processor is likely to access in the near future. Caches are useful to bridge the speed mismatch between the processor and the main memory. You can find a more detailed lecture notes here.
